# BookingHub - Projeto de Banco de Dados

## Descrição
O BookingHub é um projeto acadêmico desenvolvido para a disciplina de Banco de Dados, com o objetivo de simular um sistema de reservas de viagens e hospedagens.

O sistema possui integração entre banco de dados PostgreSQL e uma API desenvolvida em Python utilizando FastAPI.

## Tecnologias utilizadas

- PostgreSQL
- Python
- FastAPI
- Git
- GitHub
- VS Code
